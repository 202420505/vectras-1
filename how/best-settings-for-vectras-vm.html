Best Settings For Vectras VM






Step 1 : Open Menu


First of All Press on on the Top Left and then Press on Settings


Step 2 : Choose the CPU you Like


First Option is to Choose the CPU you Like Remember that it Does not Affect the Performance

It Just Give You More Instructions Sets Support So More Roms Could be Supported


Step 3 : Choose The Amount of Ram You Would Like to Use


First Thing you Should do is to Enable Custom Ram Option or You can Keep It Disabled if you Want to Vectras to Calculate How Much Does It Need

Automatically But If You Want to Specify How much Ram you Need Just Enable Custom Ram Option

Type the amount of Ram you Need but Remember to make Sure that the Amount of Ram you Choose Must Not Exceed 60% of Your Phone's Ram


Step 4 : Choose the Number of CPU Cores You Would Like to Use


Before Doing Anything First Thing you Must do is to Enable MTTCG

Then in CPU Cores Section Choose the Number of CPU Cores You Would Like to Use

Recommended to go With 2, 4 or 6 But DO NOT GO WITH 8
